03 / MULTI-AGENT NETWORK

四大 AI-Agent 协作网络

通过四大智能体跨岗位协同，完成从发现异常到自动执行、最终复盘的完整数据闭环

AGENT A
🔬
门店诊断智能体
数据诊断 · 看数据
解决"哪些门店存在异常"的敏锐感知问题，打破人工导表周度出数的滞后模式
  • 比历史：对比单店历史各周期曲线
  • 比兜底线：计算兜底风险对冲倒计时
  • 比行业：同等规模同业态均值对比
  • 比领先者：Top 5 销冠店天花板差距
  • 比区域/同城/同车型细分对比
输出：红黄绿门店报表 + 兜底风险预警报表
AGENT B
🧠
决策归因智能体
短板归因 · 做策略建议
对红黄店进行"多维钻取"，厘清业绩下滑底层真因，匹配策略模板输出建议
  • 业务维度下钻：延保/安心包/电池延保拆解
  • 车型维度下钻：特定车型话术普及分析
  • 人员行为交叉校验：ASR + LLM 智能质检
  • 考勤/提单/Pad 活跃度交叉比对
  • 区分"能力短板"vs"态度问题"
输出：自动化策略报告 + 历史决策库匹配建议
AGENT C
📦
策略执行智能体
任务派发 · 看反馈
将管理策略自动转化为系统内"策略分发与执行闭环链"，消除信息传声筒问题
  • 智能工单自动推送至 App 端
  • 支持一键采纳或自动派发
  • 带教强留痕：门店定位 + 录音上传
  • AI 智能质检带教录音内容
  • 结果关联绑定：订单号/通关考试
输出：闭环工单 + 数字化留痕审计记录
AGENT D
📈
效果追踪智能体
效果采集 · 再看效果
工单执行后长周期采集反馈数据，评估策略真实有效性，反哺组织决策
  • 3天/1周/1月短长期效果追踪
  • 红黄店摘帽变绿监测
  • AI vs 人工决策吻合度比对
  • 人员效能红黑榜
  • 带教任务转化率评估
输出：人效看板 + 汰换调岗依据链
04 / FOUR-LAYER ANALYSIS

纵深拆解：四层递进式智能分析闭环

从数据诊断到组织决策，层层深入完成"发现异常 → 自动执行 → 最终复盘"的完整闭环

01
第一层
自动化数据诊断与"红黄绿店"动态跑批
Agent A · 看数据

系统每日/周自动调取底层数仓数据，对所有合作新能源门店执行全方位动态跑批模型，打破依赖人工导表、周度出数的滞后模式

🔍 五比分析跑批法
比历史 · 比兜底线 · 比行业 · 比领先者 · 比区域/同城/同车型，全方位动态扫描
🚦 红黄绿动态分类
🟢 绿店指标健康 · 🟡 黄店趋势下滑逼近红线 · 🔴 红店触发严重信号激活归因
02
第二层
短板多维归因与决策模板套用
Agent B · 做策略建议

Agent A 筛选出红黄店后，Agent B 随即介入进行"多维钻取"，厘清业绩下滑底层真因，给出针对性策略建议

🔎 多维下钻分析
业务维度（延保/安心包/电池延保拆解）+ 车型维度（新车型话术普及）+ 人员行为交叉校验（ASR + LLM 智能质检）
📚 决策知识库套用
遍历历史决策记录知识库，匹配最佳策略模板。区分"能力短板"（话术带教）vs"态度问题"（风控稽查）
03
第三层
"策略任务化"全流程 PDCA 工单执行
Agent C · 看反馈

将口头化、纸面化的管理策略自动转化为系统内"策略分发与执行闭环链"，彻底消除信息传递的传声筒问题

📱 智能工单自动推送
线上运营一键采纳或自动派发，将策略瞬间转化为"帮带工单"或"风控稽查任务"，直达区域主管/HRBP App 端
🔒 过程管理黑盒破解
带教强留痕（定位+录音上传）→ AI 智能质检（识别实质指导动作）→ 结果关联绑定（订单号/通关考试），未达标不予结单
04
第四层
短长期效果追踪与组织决策辅助
Agent D · 再看效果

工单执行完毕后，长周期不间断采集反馈数据，评估策略方案真实有效性，直接反哺 HRBP 团队支撑减员增效

📊 全周期效果采集
3天/1周/1月关键业绩指标追踪 · 红黄店摘帽变绿监测 · AI vs 人工决策吻合度比对 · 一键采纳反哺模型
🏢 组织优化决策辅助
人员效能红黑榜 · 带教任务转化率评估 · 汰换调岗依据链一键输出 · 支撑 90→70 人效优化目标
